# Bridge Bot Analyzer (BBA)

<p align="center">
  <img src="BBALogo.jpg" alt="Bridge Bot Analyzer Logo" width="130">
</p>

<p align="center">
  <strong>Bridge Bot Analyzer</strong> is a native Windows application for contract bridge analysis based on EPBot technology.
</p>

---

## Features

- Analysis of contract bridge deals
- Evaluation of bidding decisions and card play
- Fast deal processing and analysis
- Designed for bridge players, testers, and bot developers

---

## Repository Contents

- Native Windows application binaries
- Configuration and integration files
- Windows COM libraries
- Windows ARM64 library
- Intermediate Language (IL) library
- Native libraries for:
  - Windows
  - Linux
  - macOS
  - WebAssembly (WASM)

---

## System Requirements

- Windows
- Supported architectures:
  - x86
  - x64
  - ARM64
- Microsoft Visual C++ Redistributable (if required)

---

## Installation

1. Download the latest release package.
2. Extract `BBA.zip`.
3. Launch the application.

---

## Running the Application

```bash
BBA.exe
```
